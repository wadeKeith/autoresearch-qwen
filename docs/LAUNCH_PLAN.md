# Launch Plan

The launch story should be:

**An agent improves `Qwen3-VL-4B` on official `DocVQA` by rewriting one training file.**

The public demo should show:

- baseline `validation` score
- one small diff in `train.py`
- improved score
- the fixed public split contract and optional blind `test` export

That is clear, compact, and easy to share.
